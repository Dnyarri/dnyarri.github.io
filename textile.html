POVRay mosaic: tori

Textile and cross-stitch simulation in POVRay

Linen program reads source PNG file and convert it into POVRay scene, simulating canvas of linen or tabby weave type (gros de Naples), having nodes colored after source image pixels, looking like print on canvas.

Global scene parameters include thread finish (dull or shiny) and randomization settings (some node randomization is used to break monotonous pattern).

Default scene includes pinhole camera with autofocus on object, which can be easily changed to any other camera type available in POVRay.

Stitch program reads source PNG file and convert it into POVRay scene, simulating counted cross stitch, with each stitch 3D object colored after source image pixel.

Thread normal texture is somewhat randomized on per-stitch basis to make rendering more realistic.

Exported embroidery POVRay scene is configurable within POVRay since as many parameters, as possible, like stitch geometry and it's randomization, are declared as global and therefore are easy to change for the whole scene.

Lights and camera grouped in POV-file for easy editing of final scene.


Proceed to POVmosaic at GitHub for downloads.
