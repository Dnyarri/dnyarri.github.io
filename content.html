After closing AmphiSoft plugins projest long ago in a universe far away The Toad of Great Mental Powers made a decision to implement some of spontaneous ideas, developed or implied, into something potentially easy to understand and hopefully simple to use by humans. As an "easy to understand" mean of production Python was chosen (that was probably an unrecoverable mistake. As a result, some Python projects described hereby were created and shared on GitHub, free to download and use.

Amble, i.e., the projects:

POVRay img2mesh

Program for converting PNG image into POVRay 3D mesh, so that bright pixels turns into high peaks on resulting 3D landscape - the more is brightness of source image pixel, the higher is 3D mesh node point. Exported POV file contains everything necessary to be rendered right out of the box (camera, lights, texture), or may be edited later. Read more...

POVRay mosaic

This project currently lists 12 programs, converting PNG image into POVRay mosaic, consisting of various objects - spheres, cylinders, boxes, hexagonal or triangular prisms, blobs or superellipsoids.

Some programs of that group create rather complex scenes, simulating colored canvas or counted cross-stitch.

Rendered images are suitable for scrapbooks, posters, and other creative ways to entertain and/or annoy people.

All program's POV output is reasonably structured to simplify further editing (changing light, camera, moving and rotating objects, changing object texture).

Scale2x and Scale3x in pure Python

Pure Python implementation of Scale2x and Scale3x image rescaling algorithms. Command line, GUI and batch processing programs available, free and open source. Read more...
