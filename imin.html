ImIn (Image interpolation)

Barycentric and bilinear image interpolation and rescaling in pure Python

ImIn (Image Interpolation) module contains Python code for bilinear image interpolation and barycentric image interpolation in general and rescaling in particular. That is, it contains functions for for reading a pixel at float coordinates, interpolated from real pixels, and separate functions for rescaling whole image; the latter optimized for speed.

Currently code is at development stage, and is supposed to be improved later.
